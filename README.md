Calendar And Leave Management Eco-System (Offline)
Project Overview
CALME is an offline Calendar and Leave Management System designed to manage employee leaves, holidays, and attendance efficiently without requiring an internet connection.
It is ideal for organizations that need a secure, local, and lightweight leave management solution.

Features
Calendar-based leave tracking

Employee leave management (offline)

Leave request & approval workflow

Leave reports & summaries

Works completely offline

Desktop-based interface

Local data storage (no cloud dependency)
